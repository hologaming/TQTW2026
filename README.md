# Titan Quest Traditional Chinese Patch

I only have the Titan Quest Anniversary Edition, so the DLC translations have not been tested (this is NOT Titan Quest II).<br> 
Game version date: August 4, 2025<br>
Translation: Translated using GPT-5.2<br>
Translation version: v1.0<br>
Translation date: March 21, 2026

## Translation Feedback

If there are any translation errors, suggestions and corrections are welcome.

## Installation
Place Text_CN.arc into the Text folder in the game directory and overwrite the original file.
Go into the game settings and switch the language to "Chinese".
Restart the game for the changes to take effect.
